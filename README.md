# WebProject
# Project Description

This project was established to develop the official Rawnaq Web Application. Rawnaq is a Saudi brand specializing in women's fashion. Currently, the brand handles its sales manually through social media like WhatsApp and Instagram, which causes some issues such as lack of immediate inventory update, possibility of errors in data recording. As a solution we designed a professional digital platform to automate Rawnaq’s shopping processes, allowing Rawnaq to make their customers shop in an easy and organized way, and so for the brand to manage their operations well.

# Technologies Used

In this project, we used simple web development technologies to build a clean and user-friendly interface. The main language used was HTML, which helped us structure the pages such as Home, Shop, Payment, and Return. We also used CSS to design and style the website, including colors, layout, and overall appearance to match the brand identity of Rawnaq,The project was also organized into multiple pages connected through a navigation bar to ensure smooth user experience.

In addition, we used basic JavaScript functions to add simple interactions, such as showing alert messages when users complete actions like payment or submitting a return request. This made the website more interactive and user-friendly. For the design phase, we used Appsmith to create the initial layout and visualize how the application should look before converting it into actual code. After that, we implemented the design using Visual Studio Code, where we wrote and organized our HTML and CSS files.

Overall, these technologies helped us create a simple, organized, and functional web application that improves the shopping experience for users.
